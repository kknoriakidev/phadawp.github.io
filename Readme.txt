Краткое описание:
=================
1. Стильный шаблон;
2. Админка записывается напрямую в users.ini (По протоколу FTP);
3. Поддержка нескольких серверов;
4. Поддержка нескольких тарифов и их сроков;
5. Админка работает по Ник + Пароль;
6. Блокировка добавления ников " . ", Player, monitor и т.д;
7. Автоматическое приостановление админок;
8. Удобная и красивая панель управления;
9. Мобильная версия;
10. Быстрая работа сайта;
11. Оплата через FREE-KASSA;
12. Вывод списка серверов на главной странице;
13. Выбор цвета всего сайта;
14. Мониторинг(для опроса серверов должны быть открыты соответствующие UDP/TCP порты);
15. Полностью открытый исходный код.
=========================================================================
Установка:
==========
1. Загрузить все файлы на хостинг
2. Установите права 0777 для следующих директорий: uploads, uploads/mapsimg
3. Создайте базу данных на хостинге и загрузите в нее таблицы из файла admin_bp.sql
4. Откройте файл  core/db.php и укажите настройки от базы-данных, так же укажите url сайта и данные для входа в админку.
